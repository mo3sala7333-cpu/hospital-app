# Project Scope

Version: 1.0

Status: Planning

---

# Project Overview

This project is a premium cinematic interactive digital experience for Al Hayat Medical Services Company.

It is designed exclusively for investor presentations and large conference displays.

The experience is not intended to function as a traditional corporate website.

Instead, it should feel like a cinematic journey through a futuristic healthcare ecosystem.

---

# Primary Audience

- Investors
- Board Members
- Business Partners
- VIP Guests
- Healthcare Executives

---

# Presentation Environment

The experience will be displayed on a large conference screen.

It should look impressive from a distance as well as up close.

The design must prioritize visual impact over dense textual information.

---

# User Interaction

The entire experience is controlled by scrolling.

There are:

- No navigation bar
- No traditional pages
- No slide presentation
- No dashboard
- No forms
- No login system

Scrolling drives the cinematic journey.

---

# Media Rules

Only real company assets may be used.

This includes:

- Real hospital buildings
- Real interior photography
- Real company logo

No fake hospitals.

No AI-generated replacement buildings.

No placeholder images.

The supplied assets are the heroes of the experience.

---

# Experience Goals

The visitor should feel:

- Trust
- Luxury
- Innovation
- Stability
- Growth
- Future Vision

Every scene must reinforce these emotions.

---

# Performance Goals

The experience must provide:

- Smooth scrolling
- Stable 60 FPS whenever possible
- Responsive layout
- Optimized image loading
- Lazy loading where appropriate
- Clean modular architecture
- Zero console errors

---

# Technical Stack

The implementation should use:

- HTML5
- CSS3
- JavaScript (ES Modules)
- GSAP
- ScrollTrigger
- Lenis
- Three.js only where it adds real value

---

# Out of Scope

The following features are intentionally excluded:

- CMS
- Admin Panel
- Authentication
- User Accounts
- Blog
- E-commerce
- Booking System
- Contact Forms

The project focuses entirely on delivering an unforgettable investor experience.
